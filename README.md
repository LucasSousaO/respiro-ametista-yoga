# Respiro Landing Page (Ametista Yoga)

## Setup
- Suba este repositório no GitHub Pages (ou Vercel/Netlify).
- Aponte o subdomínio `respiro.ametistayoga.com.br`.

## Ajustes obrigatórios
1) Trocar em `index.html`:
- SEU_DOMINIO
- SEU_FORM_ID (Google Forms)

2) Trocar imagens em `assets/img/`:
- hero.jpg
- og-cover.jpg (1200x630)
- favicon.png (32x32)

## Tracking
Eventos via `dataLayer`:
- menu_open
- cta_click
- modal_open / modal_close
- scroll_depth
- time_on_page

Pronto para GTM/GA4.
